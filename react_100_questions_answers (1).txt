1. What is React?
React is a JavaScript library for building user interfaces using component-based architecture.

2. What are the features of React?
Virtual DOM, component-based, one-way data binding, hooks, reusable components.

3. What is JSX?
JSX is a syntax extension that allows writing HTML-like code in JavaScript.

4. How does JSX work?
JSX gets transpiled into React.createElement() calls using Babel.

5. What is the virtual DOM?
A lightweight copy of real DOM used to improve performance by minimizing updates.

6. How does React differ from Angular?
React is a library focused on UI, Angular is a full framework.

7. What are components in React?
Reusable building blocks of UI.

8. Functional vs class components?
Functional use hooks, class use lifecycle methods.

9. What are props?
Inputs passed to components.

10. What is state?
Local data managed inside component.

11. What is useState?
Hook to manage state in functional components.

12. What is useEffect?
Hook to handle side effects like API calls.

13. Lifecycle of React?
Mounting, Updating, Unmounting.

14. Controlled components?
Form elements controlled by React state.

15. Uncontrolled components?
Form elements managed by DOM.

16. What is prop drilling?
Passing props through many levels.

17. How to avoid prop drilling?
Use Context API.

18. What is Context API?
Global state sharing without props.

19. What is Redux?
State management library.

20. Redux vs Context?
Redux for large apps, Context for simple global state.

21. What is React Router?
Library for navigation.

22. What is SPA?
Single Page Application.

23. What are hooks?
Functions to use state/lifecycle in functional components.

24. What is useRef?
Access DOM or persist values.

25. What is useMemo?
Optimize expensive calculations.

26. What is useCallback?
Memoize functions.

27. What is lazy loading?
Load components when needed.

28. What is code splitting?
Split code into chunks.

29. HOC?

30. Render props?
Sharing code using function prop.

31. Reconciliation?

32. Keys?

35. Fragment?

36. StrictMode?

37. Error boundary?
